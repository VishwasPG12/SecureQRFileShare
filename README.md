# Secure File Sharing with QR-based Key Exchange

This project demonstrates AES-256 encryption and QR-based offline key sharing using Streamlit.

## Features
- Encrypt any file (PDF, DOCX, images, etc.)
- AES-256-GCM encryption
- QR code-based secure key sharing
- Decrypt encrypted files using scanned key
- No internet-based key transmission

## How to Run
pip install -r requirements.txt
streamlit run app.py
